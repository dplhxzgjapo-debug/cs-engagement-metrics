# プロジェクト概要

ポケモンチャンピオンズ（シングルバトル）の対戦リファレンスツール。
GitHub Pagesでホスティングし、スマホブラウザから参照する。

## 構成
- `index.html` — 単一HTMLファイル（CSS/JS内蔵、外部依存なし）

## 機能
1. **選出アシスタント** — 相手のポケモンをタップ選択→最適な選出パターンを自動提案
2. **パーティ一覧** — 自分の6体の技・持ち物・役割
3. **素早さ早見表** — 自チーム vs 環境上位の比較
4. **環境ポケモン対策** — TOP20の処理ルート
5. **タイプ相性表**

## 自分のパーティ（メガゲンガー軸サイクル）
メガゲンガー / アーマーガア / アシレーヌ / カバルドン / ガブリアス / サザンドラ

## デプロイ
mainブランチにpushすればGitHub Pagesに自動反映。
URL: https://dplhxzgjapo-debug.github.io/cs-engagement-metrics/

## データ更新
POKEDB配列（環境ポケモンDB）とMY_TEAM配列を編集する。
使用率の変動や新しい型の流行があれば適宜更新。
