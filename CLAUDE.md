# プロジェクト概要

ポケモンチャンピオンズ（シングルバトル）の対戦リファレンスツール。
GitHub Pagesでホスティングし、スマホブラウザから参照する。

## 構成
- `index.html` — 単一HTMLファイル（CSS/JS内蔵、外部依存なし）

## 機能
1. **選出アシスタント** — 相手のポケモンをタップ選択→最適な選出パターンを自動提案
2. **パーティ一覧** — 自分の6体の技・持ち物・役割
3. **素早さ早見表** — 自チーム vs 環境上位の比較
4. **環境ポケモン対策** — TOP20の処理ルート
5. **タイプ相性表**

## 自分のパーティ（メガゲンガー軸サイクル）
メガゲンガー / アーマーガア / ウォッシュロトム / カバルドン / ガブリアス / サザンドラ

変更履歴:
- アシレーヌ → ウォッシュロトム (2026-05-05)
  - 理由: マフォクシー/ブリジュラス/リザードンへの構造的弱点解消
  - 失ったもの: フェアリー打点（妖4倍/2倍取れる相手は受け回しで対応）

## 制約（重要・必ず守る）
- **ポケモン提案・技・持ち物はすべて、ポケモンチャンピオンズの内定範囲内に限る**
- 内定ポケモンのリストは `data.json` の `all_pokemon`（187体）
- ヒードラン・ランドロス・カプ・テッカグヤ等は内定外なので提案しない
- 技構成・持ち物もチャンピオンズで使えるものだけを提案する
